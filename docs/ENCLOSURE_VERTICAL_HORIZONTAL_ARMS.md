# 垂直挟み込み（列固定）— 水平腕と Pseudo（`board-worker.js`）

## 画面上の向き

- **垂直挟み込み**の破線＝**列 `nc` 固定**（画面上では縦線）。
- **水平腕**＝**行 `y` 固定**で `x` が変わる辺。`dx > 0` を **R**、`dx < 0` を **L**（`path`: `{ x: 列, y: 行 }`）。

## 囲い込み成立（垂直）

列上の最小行・最大行のブラケット `iMin` / `iMax`（従来どおり）のうえで、

- **上ブラケット**: 画面上の水平腕を **R を優先**、なければ L。
- **下ブラケット**: **L（東から入る -x）を優先**、なければ R。

両腕の符号が **互いに逆**（積が -1）なら成立。`same_horizontal_arm_direction` のとき **パス反転**でもう一度試す。

## Pseudo（垂直のとき）

- **下ブラケット**に、**同じ行で列がブラケットより東**の隣接マスがある場合のみ:
  - ブラケットを出た **続きのパス**について **最大 4 辺**を辿り、**画面上水平で R（+x）** の辺があれば pseudo 棄却。
- 上ブラケット側の水平 L 探索は、西に伸びる幹ですぐ真になりやすいため **未実装**（必要なら東西対称ルールを別途）。
